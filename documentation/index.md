# Maint — CMMS Knowledge Bundle

This bundle documents the architecture, module specifications, and domain model of **Maint**, a multi-tenant Computerized Maintenance Management System powered by **Vue.js 3** (frontend) and **Spring Boot** (backend).

## Structure

- [`architecture.md`](architecture.md) — System architecture (layers, deployment, data flow)
- [`specs/index.md`](specs/index.md) — Module specification index
- [`specs/modules/`](specs/modules/) — Per-module specifications with entity and state diagrams
- [`ui-design/index.md`](ui-design/index.md) — UI design system, layout, navigation, page wireframes
- [`ui-design/modules/`](ui-design/modules/) — Per-module page specifications with wireframe diagrams
- [`references/`](references/) — External references (OKF spec, standards)

## Conformance

This bundle conforms to [OKF v0.1](references/okf-spec.md).
